Elemen HTML: <audio>

<audio> adalah elemen HTML yang digunakan untuk menyematkan audio ke dalam halaman web.

Anda dapat menggunakan atribut seperti src untuk menentukan lokasi file audio dan controls untuk menampilkan kontrol pemutaran audio di halaman web.

Contohnya, <audio src="audiofile.mp3" controls></audio> akan menampilkan pemutar audio untuk file "audiofile.mp3".
